Bem-vindo à Engenharia de Prompt

O sistema operacional para a mente do arquiteto do futuro.

Como Navegar o Curso

Use o menu à esquerda para navegar entre os módulos e aulas. Seu progresso é salvo automaticamente a cada aula que você marca como concluída.

Cada aula possui um arsenal com materiais de apoio e um campo para suas anotações pessoais, que ficam salvas no seu navegador.

Ao final de cada módulo, você desbloqueará uma conquista, que pode ser vista no Painel de Conquistas.
